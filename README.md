# Easy Douyin
一个极简版的抖音服务器，包含登录，注册，发布视频，首页刷新视频，关注取关用户等功能。

# Dependency
**开发前的配置要求**：
- go 1.18.1
- MySQL
- 搭建Redis、RabbitMQ环境
- 配置静态资源服务器：安装Nginx、vsftpd、ffmpeg
- [最新版抖音客户端软件](https://bytedance.feishu.cn/docs/doccnM9KkBAdyDhg8qaeGlIz7S7)
